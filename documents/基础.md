布尔值
```
    let isShow: boolean = false

    let isShow: boolean = new Boolean(1) 
    // Type 'Boolean' is not assignable to type 'boolean' 

    let isShow: Boolean = new Boolean(1)

    let isShow: boolean = Boolean(1)

    注： 
    1. 使用构造函数Boolean构造出的是一个Boolean对象
    2. 直接调用Boolean返回的是一个boolean类型
```
